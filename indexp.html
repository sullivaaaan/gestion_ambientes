1. Introducción

Este proyecto busca desarrollar un sistema de control de accesos para los ambientes del SENA, usando métodos avanzados de autenticación. El sistema garantizará seguridad, control eficiente y registro de tiempos.

2. Abstract

This project aims to develop an advanced access control system tailored for SENA's environments. Utilizing sophisticated authentication methods, the system provides enhanced security and efficient management of entry permissions. It will enable real-time monitoring and recording of access times, ensuring authorized personnel can seamlessly enter restricted areas while keeping detailed logs for administrative purposes.

3. Descripción del Proyecto

El Sistema de Control de Accesos para el SENA es una solución tecnológica diseñada para listar los ambientes y dar acceso únicamente a personas autorizadas. Utiliza dispositivos electrónicos y métodos de autenticación avanzados para asegurar el ingreso correcto en tiempo real.

4. Objetivos

Objetivo General

Implementar un sistema eficiente para la reserva y administración de ambientes estudiantiles, optimizando el uso de espacios y garantizando asignación equitativa.

Objetivos Específicos

  • Optimizar la gestión de ambientes con plataforma digital en tiempo real.
  • Automatizar asignación según capacidad, uso y horarios.
  • Mejorar accesibilidad y transparencia para usuarios.
  • Implementar controles y reportes para evaluar eficiencia.

Justificación

El sistema permite listar los ambientes y dar acceso solo a personas autorizadas mediante reconocimiento facial. Este proyecto permitirá garantizar el acceso correcto, reducir riesgos de pérdida o daño de recursos y facilitar la administración.

7. Alcance

Automatizar

El sistema permitirá automatizar el listado de ambientes y el control de accesos, asegurando que solo personas autorizadas puedan ingresar, reduciendo la intervención manual y los errores asociados.

Facilitar

Se busca facilitar el acceso a los ambientes listados mediante autenticación avanzada, permitiendo a los usuarios autorizados ingresar de manera sencilla y segura.

Generar

El sistema generará reportes detallados sobre el acceso a los ambientes listados, proporcionando información clave para la optimización de recursos y la toma de decisiones.

Integrar

El alcance incluye la integración del sistema de listado y acceso con otras plataformas institucionales, permitiendo la interoperabilidad y mejora continua de los procesos.

8. Tecnologías

PHP: Programación Backend

**Puntos Fuertes:** Lenguaje del lado del servidor, ideal para desarrollo web dinámico, compatible con bases de datos, y cuenta con una gran comunidad y frameworks robustos.

**Uso en el proyecto:** Se usaría para la lógica de negocio, la gestión de usuarios, y la interacción con la base de datos para manejar el stock y los pedidos.

SQL: Gestión de Bases de Datos

**Puntos Fuertes:** Lenguaje estándar para gestionar bases de datos relacionales, permite crear, leer, actualizar y eliminar datos de forma eficiente, asegurando la integridad de los mismos.

**Uso en el proyecto:** Fundamental para la creación, consulta y manipulación de la base de datos que almacenará información de productos, inventario, ventas, categorías y proveedores.

HTML: Estructura Web

**Puntos Fuertes:** Es el lenguaje fundamental para crear la estructura y el contenido de las páginas web. Fácil de aprender y universalmente soportado por los navegadores.

**Uso en el proyecto:** Se utiliza para definir la estructura de todas las páginas de la aplicación web, organizando el texto, las imágenes, y los elementos interactivos.

CSS: Estilo y Diseño

**Puntos Fuertes:** Lenguaje para estilizar y dar diseño a las páginas web. Permite controlar la apariencia visual, colores, fuentes, espaciado y disposición de los elementos.

**Uso en el proyecto:** Fundamental para darle una apariencia atractiva y coherente a la interfaz de usuario de la aplicación web, asegurando una buena experiencia visual para el usuario.

JavaScript: Interactividad Web

**Puntos Fuertes:** Permite añadir interactividad y dinamismo a las páginas web. Es esencial para crear experiencias de usuario ricas y manipular el contenido del lado del cliente.

**Uso en el proyecto:** Se usaría para manejar la lógica del lado del cliente, como validación de formularios, animaciones, peticiones asíncronas a la API y actualizaciones dinámicas de la interfaz.

Python: Procesamiento de Datos y Lógica

**Puntos Fuertes:** Lenguaje de propósito general, muy legible y versátil, con una vasta colección de librerías para ciencia de datos, IA, desarrollo web y automatización.

**Uso en el proyecto:** Podría usarse para scripts de backend, análisis de datos de inventario o ventas, o para implementar la lógica de control si se extiende a un sistema IoT más complejo.

MySQL: Base de Datos Relacional

**Puntos Fuertes:** Es un sistema de gestión de bases de datos relacional (RDBMS) popular, de código abierto, altamente confiable, escalable y compatible con SQL.

**Uso en el proyecto:** Será la base de datos principal para almacenar toda la información del sistema, incluyendo productos, categorías, proveedores, movimientos de inventario y ventas.

Protoboard: Prototipado Electrónico

**Puntos Fuertes:** Placa de prototipado que permite construir y probar circuitos electrónicos de forma temporal sin necesidad de soldadura. Es reutilizable y facilita la experimentación.

**Uso en el proyecto:** Se utilizará en la fase de desarrollo del hardware para montar y probar los circuitos con Arduino y los componentes electrónicos antes de una implementación final.

Componentes Electrónicos: Base del Hardware

**Puntos Fuertes:** Son los bloques de construcción de cualquier circuito electrónico (resistencias, condensadores, sensores, LEDs, etc.), cada uno con una función específica y esencial.

**Uso en el proyecto:** Se utilizarán para construir la parte física del sistema (sensores, actuadores, módulos de comunicación) que interactuarán con los ambientes a controlar.

Word: Procesador de Textos

**Puntos Fuertes:** Herramienta ampliamente utilizada para la creación y edición de documentos de texto con formato, incluyendo tablas, imágenes y gráficos.

**Uso en el proyecto:** Se emplea para la elaboración de la documentación del proyecto, informes, manuales de usuario y cualquier otro documento textual necesario.

Excel: Hojas de Cálculo y Análisis

**Puntos Fuertes:** Aplicación de hojas de cálculo potente para organizar, analizar y visualizar datos numéricos, realizar cálculos complejos y crear gráficos.

**Uso en el proyecto:** Puede utilizarse para la gestión inicial de datos, planificación de recursos, o para un análisis básico de datos exportados de la base de datos.

Visual Studio Code: Editor de Código

**Puntos Fuertes:** Editor de código fuente ligero y potente, compatible con una gran variedad de lenguajes de programación y ofrece extensiones para ampliar funcionalidades, depuración y control de versiones.

**Uso en el proyecto:** Será el entorno de desarrollo principal para escribir y editar el código de PHP, HTML, CSS, JavaScript y Python, facilitando la programación y depuración.

Estas tecnologías habilitan la programación, gestión de datos y control hardware. Integran sistemas digitales y electrónicos para el control eficiente de ambientes.

12. Recomendaciones para Mejorar el Sistema

  • Migrar a versión más reciente de PHP: Mejorará seguridad y rendimiento.
  • Contratar máquina virtual: Optimiza la ejecución y escalabilidad del sistema.
  • Implementar módulo de inventario: Control eficaz de equipos y recursos.
  • Incorporar chat online: Facilita comunicación rápida entre usuarios.
  • Utilizar Raspberry: Para control físico de accesos y sensores.
